📋 1. Quelles données collectons-nous ?

✅ Ce qu'on collecte (presque rien)

  • Ton thème préféré — sauvegardé localement
  • Tes sauvegardes (CV, fiches, quiz, objectifs) — stockées uniquement dans ton navigateur

🚫 Ce qu'on ne collecte JAMAIS

  • Ton nom, prénom, email
  • Tes notes scolaires
  • Tes informations financières
  • Ton adresse IP, localisation, historique de navigation

🔒 Tous les calculs (moyenne, budget, bourse) se font directement dans ton navigateur. Aucune donnée n'est envoyée sur un serveur.

🍪 2. Utilisation des cookies

On utilise un seul cookie : celui qui sauvegarde ton thème de couleur préféré.

🍪 Cookie "selectedTheme" — expire jamais — contient juste le nom du thème choisi.

On n'utilise pas : cookies publicitaires, cookies de tracking, cookies tiers.

💾 3. Stockage local (localStorage)

  • Les données restent sur ton appareil
  • On ne peut pas les voir
  • Tu peux les supprimer en vidant le cache
  • Tu peux les exporter via Mes Sauvegardes

⚠️ Si tu changes d'appareil ou de navigateur, tes données ne te suivent pas automatiquement. Pense à les exporter/importer !

🔐 4. Sécurité de tes données

  • Garde ton navigateur à jour
  • Utilise un mot de passe pour verrouiller ton ordinateur
  • Exporte régulièrement tes sauvegardes

📜 5. Tes droits (RGPD)

  • Droit d'accès : toutes tes données sont visibles dans Mes Sauvegardes
  • Droit de suppression : vide le cache ou utilise "Tout supprimer" dans Mes Sauvegardes
  • Droit de portabilité : export/import dans Mes Sauvegardes
